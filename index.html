Rule-Based Marathi Morphological Analyser

Developed a rule-based morphological model for Marathi using Python. Implemented suffix-mapping for verb conjugation and noun inflection, designed pluralization and tense-based transformation functions, and tested accuracy using controlled input-output validation.

Python Google Colab Rule-Based Modelling
NLP Evaluation
Human vs Machine Translation Evaluation

Genre-based comparative evaluation of human and machine translation across literary, legal-political, and conversational corpora. Built a Python pipeline for parallel corpus processing and applied METEOR and BERTScore metrics to assess fluency, stylistic retention, and cultural equivalence.

Python METEOR BERTScore Corpus Processing
Language Documentation
Phonological & Morphological Analysis of Ujoni Assamese

Field-based documentation and structural analysis of a low-resource language variety. Collected and curated 50+ hours of primary speech data, performed IPA transcription and multi-tier annotation using ELAN, and conducted acoustic phonetic analysis using Praat.

ELAN Praat IPA Transcription Corpus Annotation
